# OrientDB-NET - `Gremlin()`

This method executes Gremlin scripts.  The return value is an `OCommandResult` object.

## Executing Gremlin Scripts

### Syntax

```
OCommandResult Gremlin(string <query>)
```

- **`<query>`** Defines the command to execute
