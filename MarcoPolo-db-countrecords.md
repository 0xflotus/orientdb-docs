# MarcoPolo - `db_countrecords()`

This function returns the number of records in a database.

## Counting Records

### Syntax

```
db_countrecords(<conn>, <opts>)
```

- **`<conn>`** Defines the database connection.
- **`<opts>`** Defines additional options.
