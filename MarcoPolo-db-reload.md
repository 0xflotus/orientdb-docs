# MarcoPolo - `db_reload()`

This function reloads the database connection.

## Reloading Databases

### Syntax

```
db_reload(<conn>, <opts>)
```

- **`<conn>`** Defines the database connection.
- **`<opts>`** Defines additional options for the function.
